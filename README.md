## Project Title

Calendar App

## Description

This project is an Android calendar application that allows users to manage their personal events and interact with friends. The app includes authentication features (login and registration), event creation and management, and the ability to add and view friends.

The application is divided into two main components:

* Client (Android application)
* Server (backend connected to a database)

The server handles data storage, authentication, and communication with the client through API requests.

## Features

* User registration and login
* Add, edit, and delete events
* View events in calendar format
* Add and manage friends
* Client-server communication
* Persistent data storage using a database

## Technologies Used

* Android SDK
* Android Studio
* REST API
* Client-Server Architecture
* Database (SQLite)

## How to Run

1. Clone the repository
2. Open the project in Android Studio
3. Start the server
4. Run the Android application on an emulator or physical device
5. Make sure the client is connected to the server

## Project Structure

* Client (Android App)

  * UI (Activities / Fragments)
  * Models
  * Adapters
  * API communication

* Server

  * API endpoints
  * Database connection
  * Authentication logic

## Notes

* The server must be running before using the application
* Internet connection is required for client-server communication
